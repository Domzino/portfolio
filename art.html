    Dissertation - Larthur X
    Rigged and animated mesh
    For my 3rd Year Dissertation module I modelled, textured, rigged and animated a 3D character. The model was created and animated in Autodesk 3DS Max, alongside Photoshop CS5 for texturing.

    Goop
    Short animated sequence
    Goop is a rough 3-minute animation, introducing the plot for a forthcoming sequel. It takes place on an alternative version of Earth, where everything is powered by a new source of fuel: goop.

    This animation was made in the first semester of my second year for the 'Anymation' module; it was a solo project which took a couple of months to create.

    Steampunk Tavern
    Low-poly steampunk tavern
    This is a piece I did for my 3D modelling module in my second year. We were given a choice of locations and themes, and I decided to pick a steampunk tavern. The whole map is made to be low poly, with a single 512x512 png UVW map shared between every object.

    Feel free to try it below: Use arrow keys to walk, shift to run, space bar to jump

    Italy
    UDK map based loosely off cs_italy
    This is a piece I did for a 3D modelling module in my second year. It was based off the Counter-Strike map 'cs_italy', and was created using a mixture of photographs I took on my holiday to Spain and copyright free images.
    UCS Game Jam 2013
    Unfinished entry to 2013 Game Jam
    This entry to the 2013 Game Jam is in the 'Art' section as it isn't a playable game. It was made using Unity 2D and features art assets made by me, and the coding was done by an external coder.
